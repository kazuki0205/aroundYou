# aroundYou バックエンド

### インストール手順
1. 依存パッケージのインストール

バックエンド:
```bash
cd server
npm install
# または
yarn install
```

2. 環境変数の設定

バックエンド (.env):
```
PORT=5001
HOTPEPPER_API_KEY=*****
```

### 起動方法
1. バックエンドの起動:
```bash
cd server
npm start  
# または
yarn start
```

アプリケーションは以下のURLでアクセス可能になります：
- バックエンド: http://localhost:5001
